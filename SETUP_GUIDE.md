# ⚡ Quick Setup Guide (5 Minutes)

**Fastest way to get started!**

---

## For Absolute Beginners

### Prerequisites Check
```bash
python --version      # Should be 3.10 or higher
git --version         # Should be available
```

Not installed? [Download Python 3.11](https://python.org) and [Git](https://git-scm.com)

---

### Installation (Copy & Paste)

#### Windows Users:
```bash
git clone https://github.com/ikartiksavaliya/credit-card-fraud-detection-system-using-deep-learning.git
cd credit-card-fraud-detection-system-using-deep-learning
python -m venv venv
venv\Scripts\activate
pip install --upgrade pip
pip install -r requirements.txt
jupyter notebook
```

#### macOS/Linux Users:
```bash
git clone https://github.com/ikartiksavaliya/credit-card-fraud-detection-system-using-deep-learning.git
cd credit-card-fraud-detection-system-using-deep-learning
python3 -m venv venv
source venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
jupyter notebook
```

---

## Next: Run Your First Notebook

1. Click on `notebooks/01_eda.ipynb`
2. Press `Shift + Enter` to run each cell
3. That's it! 🎉
